# Changelog

## [v0.1.0] - 2025-01-21

### Added
- Initial release of the project.
- Implemented core features for interacting with Last.fm API:
  - **Album**
  - **Auth**
  - **Artist**
  - **Chart**
  - **Geo**
  - **Library**
  - **Tag**
  - **Track**
  - **User**

### Notes
- The **User** endpoint currently does not include full scrobbling functionality.
